# Smart Lender - Loan Approval Prediction

## Project Description
Smart Lender is a Machine Learning web application developed using Python and Flask. It predicts whether a loan will be approved or rejected based on applicant details.

## Features
- Loan Approval Prediction
- User Friendly Interface
- Background Image
- Bank Logo
- Machine Learning Model
- Flask Web Application

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Scikit-learn
- XGBoost
